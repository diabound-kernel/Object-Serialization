# Object-Serialization
YAML/JSON object serialization using compile-time introspection

Dependencies:
  * C++20
  * CMake
  * {fmt}
  * nlohman::json
  * yaml-cpp

Install:

From the source directory, run
  
    mkdir build
       
    cd build
       
    cmake ..

  and then the usual

    make
  
    make install
